# React Components Example

This page demonstrates how to use **custom React components** in Docusaurus.

## Live Gift Gallery

import GiftGallery from '@site/src/components/GiftGallery';

<GiftGallery />

## Using Individual Cards

You can also use the `GiftCard` component directly:

```tsx
import GiftCard from '@site/src/components/GiftCard';

<GiftCard
  id="my_gift_001"
  name="My Custom Gift"
  description="A special gift just for you!"
  rarity="epic"
/>
```

import GiftCard from '@site/src/components/GiftCard';

<GiftCard
  id="custom_001"
  name="Custom Epic Gift"
  description="This is a custom epic NFT gift created inline!"
  rarity="epic"
/>

## Component Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `string` | required | Unique gift identifier |
| `name` | `string` | required | Display name |
| `description` | `string` | required | Gift description |
| `imageUrl` | `string` | placeholder | Image URL |
| `rarity` | `'common' \| 'rare' \| 'epic' \| 'legendary'` | `'common'` | Rarity level |

## Interactive Features

- **Click** on any card to flip it and see the description
- Cards have different **gradient borders** based on rarity
- Smooth **3D flip animation** using CSS transforms
