# Работа с Service Bot

Наш бот в Telegram: [Service Bot](https://t.me/)

1. При переходе в бота, нажмите кнопку `Старт`.
2. Выбери тариф, который хотите использовать и купите за звезды Вашего аккаунта Telegram (после покупки Вашему аккаунту присвоится **ключ лицензии**).
